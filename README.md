Sistema de Controle da Livraria
===============================

Funcionalidades Principais
--------------------------

1. Pacote: *Manter Estoque*

    * Registrar Venda
    * Registrar Pedido de Venda
    * Conferir Estoque
    * Ajustar Estoque
    * Definir Precos
    * Registrar Recebimento de Pedido
    * Configurar Venda
    * Configurar Compra
    * Registrar Compra
    * Incluir Livro
    * Selecao de Livro



2. Pacote: *Manter Caixa*

    * Abrir Caixa
    * Realizar Sangria
    * Realizar Suprimento
    * Fechar Caixa
    * Configurar Caixa
